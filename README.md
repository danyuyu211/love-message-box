# Love Message Box 💌

A sweet interactive love confession page — open in any browser, no installation needed.

**🔗 Live Demo:** [danyuyu211.github.io/love-message-box](https://danyuyu211.github.io/love-message-box/)

## ✨ Features

- **Q&A Card** — "Do you like me?" with a playful "No" button that runs away
- **Heart Wall** — Cards assemble into a heart shape, then scatter randomly across the screen
- **Click to Enlarge** — Tap any card to zoom in and read the message
- **Falling Particles** — Hearts and stars float in the background
- **Loading Animation** — CSS heart with progress bar on startup
- **Mobile & Desktop** — Responsive design, portrait lock on mobile

## 🚀 Quick Start

Open `index.html` in any browser. No server, no dependencies.

## 📁 Files

- `index.html` — Complete app (CSS + JS inline, sticker image base64-embedded)
- `sticker.png` — Source image asset

## 📱 Compatibility

WeChat, Safari, Chrome, QQ Browser — works offline.
